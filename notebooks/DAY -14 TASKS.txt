Task 2 + Task 3 Workflow

Client sends request
        ↓
FastAPI receives JSON body
        ↓
Pydantic validates request fields
        ↓
┌───────────────────────────────┐
│ If request is invalid         │
│ → Return custom 422 error     │
│ → Stop processing             │
└───────────────────────────────┘
        ↓ valid request
Endpoint runs prediction logic
        ↓
Preprocess input data
        ↓
Model generates churn probability
        ↓
Compare probability with threshold
        ↓
Assign label (0 or 1)
        ↓
Map probability to risk_band
(low / medium / high)
        ↓
Build response object
        ↓
FastAPI validates response_model
        ↓
Return final JSON to client
