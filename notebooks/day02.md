# Day 2 – Dataset Selection

## Dataset chosen
subscription_service_churn_dataset_real-2.csv

## Basic confirmation
- Target column: Churn
- ID column: CustomerID
- Total columns: 21

## Notes
This dataset is suitable for a streaming/subscription churn prediction project.
It includes customer subscription details, payment method, viewing behavior, engagement-related features, and a churn label.

## Example feature groups
- Subscription features: SubscriptionType, PaperlessBilling
- Payment features: PaymentMethod, MonthlyCharges, TotalCharges
- Viewing behavior: ViewingHoursPerWeek, AverageViewingDuration, ContentDownloadsPerMonth
- Preference/profile features: GenrePreference, DeviceRegistered, Gender
- Engagement/support: WatchlistSize, SupportTicketsPerMonth
